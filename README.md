# FutureForge-AI


FutureForge AI is an educational career-planning assistant designed to help students transform ambitious goals into clear and achievable action plans.

The application analyzes a student's career interests, current experience, and available study time to generate a personalized roadmap, skill recommendations, scholarship readiness insights, and daily learning plans.

I built this project because many talented students know what they want to achieve but struggle to identify the steps required to get there. Through FutureForge AI, I wanted to create a tool that empowers students to make informed decisions, stay motivated, and pursue opportunities with confidence.

This project combines problem-solving, user-centered design, and educational impact while reflecting my passion for helping students unlock their potential through technology.
